# For My Mochi 🍡💕

A cute little one-page site made for Mochi.

## How to publish it on GitHub Pages (takes 2 minutes)

1. Create a **new repository** on GitHub (e.g. `for-mochi`). It can be public or private — GitHub Pages needs public on the free plan (or a paid plan for private).
2. Upload **all the files in this folder** (`index.html`, `style.css`, `script.js`, `photo.jpg`, this `README.md`) straight into the **root** of that repository — don't put them inside a subfolder.
   - Easiest way: on the repo page, click "Add file" → "Upload files", drag all the files in, and commit.
3. Go to the repo's **Settings → Pages**.
4. Under "Build and deployment" → "Source", choose **Deploy from a branch**.
5. Pick branch `main` (or `master`) and folder `/ (root)`, then Save.
6. Wait about a minute, then GitHub will give you a link like:
   `https://yourusername.github.io/for-mochi/`
7. Open it — that's your live page! Share the link with her. 💗

## Want to tweak anything?

- **Dates / counters**: open `script.js`, edit the two dates at the top (`TALKING_SINCE` and `COUPLE_SINCE`).
- **Text / letter**: open `index.html` and edit the wording directly — it's all plain readable text.
- **Photo**: replace `photo.jpg` with any other picture (keep the same file name, or update the `src` in `index.html`).
- **Colors**: open `style.css`, the main colors are defined at the very top under `:root`.

No coding knowledge needed beyond copy-pasting text. Enjoy, and good luck 🍡
